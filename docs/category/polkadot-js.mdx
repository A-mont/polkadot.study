# polkadot.js

Polkadot does JavaScript. [polkadot.js](https://github.com/polkadot-js) provides many handy tools to interact with polkadot / substrate nodes. 
It e.g. offers [APIs](https://github.com/polkadot-js/api) around Polkadot and Substrate based chains via RPC calls. The github orginazation is also root of the polkadot.js browser extension,
hosts, the public polkadot apps repository and more.

## Official Documentation / Repositories

- [Official polkadot{.js} docs](https://polkadot.js.org/docs/): The official docs with lots of code examples and best practices.
- [Polkadot JS Apps](https://github.com/polkadot-js/apps): The repository for the official Polkadot JS user interface that showcases a lot of usage examples of the library.
